# MultiLoginBrowser - Release Notes

## Installation

Download **MultiLoginBrowser-win-x64.zip** from the release assets and extract it to a local folder.

Run:

```cmd
MultiLoginBrowser.exe
```

The release is an unpackaged Windows x64 build. It keeps browser profile data outside the repository and does not store account passwords in the application.

## What's included

| Component | Description |
|-----------|-------------|
| MultiLoginBrowser | .NET MAUI Windows app for opening multiple isolated browser profiles |
| WebView2 isolation | Separate WebView2 user data folders per profile on Windows |

## System requirements

| Requirement | Minimum |
|-------------|---------|
| Operating system | Windows 10 or Windows 11, x64 |
| WebView2 Runtime | Microsoft Edge WebView2 Runtime |

For full documentation see the source repository README.
